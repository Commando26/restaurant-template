# Restaurant Template — Project Context for Claude Code

## What This Is
A reusable restaurant website template built as a side hustle product. The idea is to find small food businesses in Oakdale, MN (and surrounding areas) that don't have websites, and offer them a professional site with a CMS they can manage themselves. Each client gets their own domain, Sanity project, and Vercel deploy.

## Tech Stack
- **Next.js 14** (App Router, TypeScript) — SSR/SSG for SEO
- **Sanity.io** (headless CMS) — restaurant owners log into Sanity Studio to edit their own content
- **Tailwind CSS** — styling with CSS variable-based theming
- **Vercel** — hosting and deploys
- **lucide-react** — icons
- **@portabletext/react** — rendering Sanity rich text (used on About page)

## Theming System
Colors are driven by CSS variables set via `[data-theme]` attribute on `<html>`. The theme value comes from Sanity (`siteSettings.colorTheme`). Available themes: warm, cool, earth, bold, minimal. Tailwind classes reference these variables (e.g. `bg-primary`, `text-text-muted`, `bg-surface-alt`).

Fonts: `font-display` = Playfair Display (headings), `font-body` = Source Sans 3 (body text). Loaded via next/font/google.

## Key Patterns
- **Data fetching**: All pages use server components with `client.fetch(query)` at the top level. `revalidate = 60` on each page for ISR (content updates within 60s).
- **Singletons**: businessInfo, operatingHours, announcement, siteSettings, aboutSection are all singleton documents in Sanity (one per restaurant). The Studio sidebar is configured to open them directly instead of showing a list.
- **Menu structure**: menuCategory has many menuItems (via reference). fullMenuQuery uses a GROQ subquery to nest items under their category.
- **Image handling**: All Sanity images go through `urlFor()` from `@sanity/image-url` with width/quality params. Next.js Image component with `fill` + `object-cover` for responsive images.

## Business Context
Target clients are small food businesses in Oakdale, MN without websites. Pricing model: $800-1500 setup + $50-75/month recurring. Per-client hosting cost is ~$1/month. Top prospects identified: La Autentica (Hispanic grocery), Bobalicious (boba café), Oak Smoke, Detailed Cleaning Services, Jokky Catering (Nigerian food).
